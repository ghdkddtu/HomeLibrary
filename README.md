# HomeLibrary - Домашняя библиотека

Веб-приложение для учета и систематизации домашней коллекции книг.

## О проекте

HomeLibrary - это простое и удобное веб-приложение, которое поможет вам:
- Каталогизировать домашнюю коллекцию книг
- Сортировать издания по категориям и жанрам
- Быстро находить нужные книги
- Отслеживать прочитанные книги

## Возможности

- **Учет книг** - Добавление, редактирование и удаление книг
- **Категории** - 11 категорий включая мангу, манхву и комиксы
- **Поиск и фильтрация** - Поиск по названию и автору, фильтрация по категориям
- **Статус чтения** - Отметка прочитанных книг
- **Локальное хранение** - Данные сохраняются в вашем браузере

## Категории книг

1. Учёба - учебники и методички
2. Работа - профессиональная литература
3. Детские - книги для детей
4. Художественная - художественная литература
5. Хобби - кулинария, рукоделие и д.р.
