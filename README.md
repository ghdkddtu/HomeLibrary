# Моя библиотека

Веб-приложение для учёта домашней коллекции книг.

## Что умеет

**Добавлять книги** — название, автор, категория, год, описание, обложка  
**Просматривать коллекцию** — все книги в виде карточек  
**Искать книги** — по названию и автору  
**Фильтровать** — по категориям, статусу чтения, году  
**Отмечать статус** — не читал/читаю/прочитано/брошено  
**Редактировать и удалять** — менять данные о книгах  

## Как выглядит

**Главная страница:**  
- Карточки всех книг с обложками  
- Кнопка "Фильтры" для поиска нужных книг  
- Кнопка "+" для добавления новых книг  

**Детальная страница книги:**  
- Вся информация о книге  
- Выпадающий список для статуса чтения  
- Кнопки "Редактировать" и "Удалить"  

## Техническая часть

**Использованные технологии:**
- Python + Flask (бэкенд)
- SQLite (база данных)
- HTML/CSS/JavaScript (фронтенд)
- Git/GitHub (контроль версий)

**Структура проекта:**
- `app.py` — основная логика приложения
- `templates/` — HTML страницы
- `static/` — стили CSS
- `library.db` — база данных с книгами

## Как запустить

1. Установи Python и Flask
2. Скачай файлы проекта
3. Запусти: `python app.py`
4. Открой: http://localhost:5000
